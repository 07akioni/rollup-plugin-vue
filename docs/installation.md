# Installation
[Node](http://nodejs.org/) and [Rollup](http://rollupjs.org) are required to use rollup-plugin-vue.

{#npm}
### [](#npm) Using NPM
```
npm install --save-dev rollup-plugin-vue
```

{#yarn}
### [](#yarn) Using yarn
```
yarn add --dev rollup-plugin-vue
```
